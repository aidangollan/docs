---
title: "Request Replacement"
description: "Request a replacement for an account"
---

## Endpoint

```
POST https://api.ugc.inc/billing/request-replacement
```

## Overview

Request a replacement for an account. The first replacement for any account is **auto-approved** â€” the old account is reclaimed and a new uninitialized account is created immediately. Subsequent replacement requests require manual admin review.

## Request Body

<ParamField body="account_id" type="string" required>
  ID of the account to replace
</ParamField>

<ParamField body="reason" type="string">
  Reason for requesting a replacement
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="Response properties">
    <ResponseField name="message" type="string">
      Status message indicating if the replacement was auto-approved or submitted for review
    </ResponseField>

    <ResponseField name="request" type="object | null">
      The billing request record

      <Expandable title="Request properties">
        <ResponseField name="id" type="string">
          Request ID
        </ResponseField>

        <ResponseField name="type" type="'replacement'">
          Request type
        </ResponseField>

        <ResponseField name="status" type="'pending' | 'approved'">
          Request status
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="new_account_id" type="string | null">
      ID of the new replacement account (only set when auto-approved)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.ugc.inc/billing/request-replacement \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"account_id": "acc_123", "reason": "Account was banned"}'
```

```typescript React
import { UGCClient } from 'ugcinc';

const client = new UGCClient({ apiKey: 'YOUR_API_KEY' });
const response = await client.billing.requestReplacement({
  account_id: 'acc_123',
  reason: 'Account was banned'
});

if (response.ok) {
  console.log(response.data.message);
  if (response.data.new_account_id) {
    console.log(`New account: ${response.data.new_account_id}`);
  }
}
```

</RequestExample>
