## Endpoint

```
POST https://api.ugc.inc/post/video
```

## Overview

Create and schedule a video post to be published on TikTok or Instagram. The video will be uploaded and posted to the specified account.

## Request Body

<ParamField body="accountId" type="string" required>
  Account ID to post from
</ParamField>

<ParamField body="videoUrl" type="string" required>
  URL to the video file (must be publicly accessible)
  
  Supported formats: MP4, MOV, AVI
  
  Recommended specs:
  - Resolution: 1080x1920 (9:16 aspect ratio)
  - Duration: 15-60 seconds
  - File size: Under 100MB
</ParamField>

<ParamField body="caption" type="string">
  Post caption/description
  
  Can include hashtags and emojis
  
  **Limit:** Maximum 500 characters
</ParamField>

<ParamField body="musicPostId" type="string">
  ID of a post that uses the song you want to attach
  
  <Note>
    **Music Support:**
    - ‚úÖ **TikTok:** Fully supported
    - ‚ùå **Instagram:** Not yet supported
  </Note>
  
  **How to use:**
  To add music to your video, provide the ID of any post that already uses the song you want. For example, if you want to use the song from this TikTok: `https://www.tiktok.com/music/The-Fate-of-Ophelia-7556815852168759313`, find a video that uses that song (e.g., `https://www.tiktok.com/@taylorswift/video/7559313483398008095`) and provide its ID: `7559313483398008095` as the `musicPostId`.
</ParamField>

<ParamField body="postTime" type="string">
  When to schedule the post (ISO 8601 format)
  
  If omitted, the post will be published immediately
  
  Example: `2024-12-31T12:00:00Z`
</ParamField>

<ParamField body="strict" type="boolean" default={false}>
  Strict mode for validation
  
  When enabled, performs additional validation checks before posting
</ParamField>

## Response

<ResponseField name="data" type="Post">
  The created post object
  
  <Expandable title="Post properties">
    <ResponseField name="id" type="string">
      Unique post identifier - use this to track the post status
    </ResponseField>
    
    <ResponseField name="account_id" type="string">
      ID of the account that will post this
    </ResponseField>
    
    <ResponseField name="type" type="'video'">
      Post type (always `video` for this endpoint)
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Initial status: `scheduled` or `pending`
    </ResponseField>
    
    <ResponseField name="social_id" type="string | null">
      Platform post ID (null until posted)
    </ResponseField>
    
    <ResponseField name="caption" type="string | null">
      Post caption
    </ResponseField>
    
    <ResponseField name="media_urls" type="string[]">
      Array containing the video URL
    </ResponseField>
    
    <ResponseField name="music_post_id" type="string | null">
      Music/audio ID
    </ResponseField>
  </Expandable>
</ResponseField>

```python Python
import requests

response = requests.post(
    'https://api.ugc.inc/post/video',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'accountId': 'acc_123456',
        'videoUrl': 'https://storage.example.com/video.mp4',
        'caption': 'My awesome video! üé• #viral #content',
        'musicPostId': 'music_xyz789',
        'postTime': '2024-12-31T12:00:00Z'
    }
)

data = response.json()

if data['ok']:
    print('Post created:', data['data']['id'])
    print('Status:', data['data']['status'])
```

```typescript React
import { UGCClient } from 'ugcinc';

const client = new UGCClient({
  apiKey: 'YOUR_API_KEY'
});

const response = await client.posts.createVideo({
  accountId: 'acc_123456',
  videoUrl: 'https://storage.example.com/video.mp4',
  caption: 'My awesome video! üé• #viral #content',
  musicPostId: 'music_xyz789',
  postTime: '2024-12-31T12:00:00Z'
});

if (response.ok) {
  console.log(`Post created with ID: ${response.data.id}`);
  console.log(`Status: ${response.data.status}`);
  
  // Monitor post status
  const statusResponse = await client.posts.getStatus({
    postId: response.data.id
  });
}
```
