---
title: "Update Media Tag"
description: "Update the tag on a media item"
---

## Endpoint

```
POST https://api.ugc.inc/media/update-tag
```

## Overview

Update the tag on a media item. Works for both user-uploaded media and social audio.

## Request Body

<ParamField body="id" type="string" required>
  The ID of the media item to update
</ParamField>

<ParamField body="tag" type="string" required>
  The new tag value
</ParamField>

## Response

<ResponseField name="data" type="UserMedia | SocialAudio">
  The updated media object
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.ugc.inc/media/update-tag \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "id": "media_abc123",
    "tag": "featured"
  }'
```

```python Python
import requests

response = requests.post(
    'https://api.ugc.inc/media/update-tag',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'id': 'media_abc123',
        'tag': 'featured'
    }
)

data = response.json()

if data['ok']:
    print(f"Updated tag to: {data['data']['tag']}")
```

```javascript JavaScript
const response = await fetch('https://api.ugc.inc/media/update-tag', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 'media_abc123',
    tag: 'featured'
  })
});

const data = await response.json();

if (data.ok) {
  console.log('Updated tag to:', data.data.tag);
}
```

```typescript React
import { UGCClient } from 'ugcinc';

const client = new UGCClient({
  apiKey: 'YOUR_API_KEY'
});

const response = await client.media.updateTag({
  id: 'media_abc123',
  tag: 'featured'
});

if (response.ok) {
  console.log('Updated tag to:', response.data.tag);
}
```

</RequestExample>
